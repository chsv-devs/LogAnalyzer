# LogAnalyzer
명령어 로그를 통하여 명령어가 실행된 횟수를 출력합니다.  

## Log format example
2021-02-23 10:05:11.001 [main] INFO  - §3§o[로그] §echzh157 §3: /명령어  
``[로그]`` 및 ``§3: /``을 기준으로 명령어 로그 여부를 판단합니다.
